Before closing a custom generation run:

- Confirm `CUSTOM_GENERATION_ARCHIVE_ROOT` points to the private companion repo.
- Save the input job description and any source URLs.
- Save prompts and agent notes.
- Save selected source facts from `content/`.
- Save generated Markdown, TeX, PDFs, and any review notes.
- Record validation, build, and privacy-scan results.
- Mark publication as `private`, `sanitized-public-example`, or `discarded`.
